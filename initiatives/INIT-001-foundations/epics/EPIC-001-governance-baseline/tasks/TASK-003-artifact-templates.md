# TASK-003 — Artifact Templates

**Epic:** EPIC-001 — Governance Baseline
**Initiative:** INIT-001 — Foundations
**Status:** Pending

---

## Purpose

Create reusable templates for the main artifact types used in the system.

Templates ensure consistent structure and enable future automation.

## Deliverables

`/templates/`

Required templates:

- `/templates/initiative-template.md`
- `/templates/epic-template.md`
- `/templates/task-template.md`
- `/templates/adr-template.md`

Each template should define the expected sections for that artifact type.

## Acceptance Criteria

- templates directory exists
- all four templates exist
- templates follow a consistent structure
